Research

I'm interested in computer vision, deep learning, image processing and machine learning. Much of my research is about image classification, segmentation and object detection. Representative papers are highlighted.

Weak-Supervised Adaptation to Detect Deepfakes in High Resolution Videos
Akash Kumar, Arnav Bhavsar
Under Review

Working on Weak Supervised Learning and Domain Adaptation to detect tampering in High Resolution Videos. Adapting from Face Forensics++ dataset (low-resolution H.264 high (40) degrees compression) to Celeb-DF dataset (high-resolution) to classify videos as original or manipulated.
Syn2Real: Forgery Classification via Unsupervised Domain Adaptation
Akash Kumar, Arnav Bhavsar
Forensics Workshop, IEEE Winter Conference on Applications of Computer Vision, 2020

Detection and localization of Copy-Move Forgery in Images. Created a tampered dataset of 80,000 images using semantic inpainting, splicing and copy-move forgery on COCO dataset. Experimenting on various Unsupervised Domain Adaptation techniques to tackle the problem of forged image classification and segmentation on CASIA dataset.
Bird Species Classification using Transfer Learning with Multistage Training
Akash Kumar, SD Das
ICVGIP Workshop, 2018

Inter-species Classification of birds in High Resolution images. Our techniques uses Mask R-CNN to crop bird images from the images. We employed Multistage-training, Transfer Learning and Model Ensembling to improve the classification accuracy on a very small dataset. (150 images)
Improving Landmark Recognition using Saliency detection and Feature classification
Akash Kumar, Sagnik Bhowmick, N Jayanthi, S Indu
ICVGIP Workshop, 2018

Devised an architecture to predict landmark labels directly from image pixels using salient regions of an image, to help better understanding and organizing photos of Indian Architectural Styles.
IceBreaker: Solving Cold Start Problem for Video Recommendation Engines
A Kumar, A Khaund, A Sharma, Y Kumar, P Kuamraguru, RR Shah, R Zimmermann
IEEE ISM Workshop, 2018

Developed a recommender system to solve the problems of ”cold-start” videos and generate a personalized recommendation based on user’s history. Worked on Data augmentation, Random Forest Regression & Deep Learning based Linear Discriminant Analysis.
Solar Potential Analysis of Rooftops Using Satellite Imagery
Akash Kumar
Technical Report

Devised an online rooftop assessment system for solar installations using Satellite Imagery for Indian cities. Implemented algorithms to identify individual rooftops of buildings and optimal area to place solar panels.
